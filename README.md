# RAVIS  Emergency Satcom device



 아두이노 Iridiumsbd.h 를 이용하여
serial 통신을 사용한 위성통신 단말기 제작 프로젝트
 



## Equipment
-
1.   Rockblock MK2
2.   Arduino Nano
3.   9V BAT -> Arduino Nano
4.   3.7V -> Rockblock MK2
5.   ESP32-CAM
6.   Battery Capacity Indicator
7.   Rottery Encoder
8.   SPST Switch
9.   RGB Module
10. 10K Register

## Language
-
C++<br/>
